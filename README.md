
# Basic Chat Website

This is a simple chat application built using Firebase  real-time database . The application allows users to join a chat room and send messages in real-time anonymously

## Features

- Real-time messaging
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
- Firebase (Realtime Database)
